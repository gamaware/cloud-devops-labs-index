# Product Overview

Cloud DevOps Labs Index is a static website and central hub for navigating educational Cloud & DevOps lab repositories organized by difficulty level (100-400, following AWS training conventions).

The site serves as:
- A professional portfolio for the author (experience, certifications, education)
- An index linking to lab repositories across domains: AWS, IaC, Containers/K8s, CI/CD, and Generative AI
- A repository access request system where users submit requests via a form that creates GitHub Issues, which are then approved/denied by the owner through GitHub Actions automation

Key features:
- Dark/light theme toggle (OneDark theme for dark mode) with localStorage persistence
- Multilingual support: English, Spanish, Portuguese with localStorage persistence
- Responsive design (desktop, tablet, mobile breakpoints)
- Hosted on GitHub Pages, deployed automatically on push to main
